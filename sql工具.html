⌨️ 快捷键指南

  • Ctrl + Enter - 在输入区域执行解析/处理操作
  • Ctrl + C - 在结果区域快速复制内容
  • Ctrl + K - 清空所有输入和结果
  • ? - 显示/隐藏此帮助对话框
  • ESC - 关闭帮助对话框

💡 使用技巧

  • 📊 实时查看数据统计信息（行数和字符数）
  • 📥 使用导出功能将结果保存为文件
  • 🔄 使用去重功能快速移除重复项
  • 🧹 使用清空按钮快速清理输入和结果
  • 🔗 JSON支持多层嵌套解析，使用"继续解析"按钮深入
